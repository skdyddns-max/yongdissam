# 구독폼 백엔드 설치 (5분)

폼 제출 → 구글 스프레드시트에 구독자 자동 저장 + 메일 알림.
티오더 랜딩에서 쓴 것과 같은 방식입니다.

## 순서

1. **새 스프레드시트 만들기** — [sheets.new](https://sheets.new) 접속, 이름은 "용디쌤 구독자" 등 자유.
2. **Apps Script 열기** — 메뉴 `확장 프로그램 → Apps Script`.
3. **코드 붙여넣기** — 기본 코드를 지우고 `Code.gs` 내용 전체를 붙여넣고 저장(💾).
4. **웹 앱으로 배포** — 우상단 `배포 → 새 배포` → 유형 `웹 앱` 선택:
   - 실행 계정: **나**
   - 액세스 권한: **모든 사용자**
   - `배포` 클릭 → 권한 승인(내 계정 선택 → 고급 → 이동 → 허용)
5. **웹 앱 URL 복사** — `https://script.google.com/macros/s/.../exec` 형태.
6. **index.html에 URL 넣기** — `const ENDPOINT = "";` 의 따옴표 안에 붙여넣기 → 저장 → 재배포(GitHub Pages면 push).

## 확인

- 페이지에서 이메일 입력 → "받아볼게요 🌱" → 스프레드시트 '구독자' 시트에 한 줄 추가 + skdyddns@gmail.com 알림 메일.
- ENDPOINT가 비어 있는 동안엔 폼이 메일 작성(mailto)으로 대신 접수됩니다 — 신청이 유실되진 않아요.

## 참고

- 같은 이메일이 두 번 신청하면 시트엔 한 번만 저장됩니다(사용자에겐 성공으로 표시).
- 코드 수정 후엔 `배포 → 배포 관리 → 수정(연필) → 새 버전`으로 재배포해야 반영됩니다.
